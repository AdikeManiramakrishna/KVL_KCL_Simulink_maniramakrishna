# KVL & KCL Circuit Simulation using MATLAB Simulink

A MATLAB Simulink project demonstrating **Kirchhoff's Voltage Law (KVL)** and **Kirchhoff's Current Law (KCL)** using a DC resistive circuit.

## Features

- MATLAB Simulink implementation
- Simscape Electrical model
- Voltage measurement
- Current measurement
- Verification of KVL
- Verification of KCL
- Educational example for students

## Requirements

- MATLAB R2025b or later
- Simulink
- Simscape Electrical

## Files

- `KVL_KCL_Circuit.slx`
- `LICENSE`
- `README.md`

## How to Run

1. Open the `.slx` file in MATLAB.
2. Ensure Simscape Electrical is installed.
3. Run the simulation.
4. Observe the voltage and current measurements.
